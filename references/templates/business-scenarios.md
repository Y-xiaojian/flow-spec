# SRM Business Scenarios

## 采购申请审批
**角色**：采购申请人、部门领导、采购员  
**流程**：申请人填写 PR -> 部门领导审批 -> 采购员执行（生成 PO 或寻源）  
**关键点**：金额阈值决定审批层级、超预算需额外审批

## 供应商准入
**角色**：寻源专员、质量、法务  
**流程**：提交供应商信息 -> 资质审核 -> 现场评审 -> 签订框架协议 -> 纳入合格供应商名录  
**关键点**：黑名单校验、资质有效期管理

## 询比价
**角色**：采购员、供应商  
**流程**：创建 RFQ -> 选择供应商 -> 发布询价 -> 供应商报价 -> 比价分析 -> 定标  
**关键点**：截止时间、密封报价、比价维度（价格、交货、质量）
