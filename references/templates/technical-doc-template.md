# 技术设计文档模板

## 模板章节

| 章节 | 说明 |
|------|------|
| 1. 背景 | 需求来源、目标、技术约束 |
| 2. 现状与调研 | 现有系统、接口或约束的摘录（人工整理；可选） |
| 3. 架构设计 | 模块划分、依赖关系 |
| 4. 接口设计 | API/函数签名、参数、返回值 |
| 5. 数据设计 | 数据模型、存储、迁移 |
| 6. 流程图 | 主流程 + 分支 |
| 7. 时序图 | 核心链路 |
| 8. 追溯表 | 需求 → 设计章节映射 |
| 9. 风险与对策 | 技术风险、备选方案 |

## 2. 现状与调研（可选）

