# SRM Glossary

## Core Concepts

| 术语 | 英文 | 定义 |
|------|------|------|
| 供应商关系管理 | SRM (Supplier Relationship Management) | 管理与供应商的全生命周期关系，涵盖寻源、合同、订单、绩效等 |
| 寻源 | Sourcing | 识别、评估、选择供应商的过程 |
| 采购 | Procurement / Purchasing | 从需求到订单、付款的采购执行流程 |
| 供应商 | Supplier / Vendor | 向企业提供产品或服务的组织 |
| 采购申请 | Purchase Request (PR) | 内部发起的采购需求单据 |
| 采购订单 | Purchase Order (PO) | 向供应商发出的正式订单 |
| 询价单 | RFQ (Request for Quotation) | 向供应商请求报价的单据 |
| 合同 | Contract | 与供应商签订的正式协议 |
| 框架协议 | Framework Agreement | 约定长期合作条款的协议 |
| 投标 | Bid / Tender | 供应商对询价或招标的响应 |

## Roles and Flows

| 术语 | 定义 |
|------|------|
| 采购员 | 执行采购操作、管理供应商日常关系的角色 |
| 审批流程 | 采购申请、合同等单据的多级审批 |
| 供应商主数据 | 供应商的基本信息、资质、银行账户等 |
| 物料主数据 | 物料编码、名称、规格、单位等 |
| 价格主数据 | 物料与供应商的约定价格、生效时间 |
